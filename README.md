# IoTSuperScale

IoT business weight application for OS 10.0.16299 and above

## Getting Started

These instructions will get you a copy of the project up and running on your local machine for development and testing purposes. See deployment for notes on how to deploy the project on a live system.

### Hardware Prerequisites

What things you need to install the software and how to install them:

1. Raspberry pi 3B+ model
2. Raspberry 7" Official touch screen (Capacitive touch)
3. Load Cell Amplifier HX711
4. An Indusrtial Load Cell bar
5. A Label printer i.e. Zebra GC420t
